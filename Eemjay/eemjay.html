There is something called SEPARATION OF CONCERNS. You should put CSS files separate and avoid having inline CSS just as you have done

You will notice the green and red background colour didn't touch the sides of your browser. This is because All browsers have something called User Agent Stylesheet.

A user agent style sheet is a ”default style sheet” provided by the browser (e.g., Chrome, Firefox, Edge, etc.). This is why I used 

* {
				box-sizing: border-box;
				margin: 0;
				padding: 0;
			}
. The code resets the user agent stylesheet and let you modify things as you wish. If you apply it to your code, you will notice that the background colors have no margins atall and look good. That code should be the first thing in your CSS file
